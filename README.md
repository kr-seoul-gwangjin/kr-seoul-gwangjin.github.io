# 광진구 · 서울특별시

> 광진구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-seoul-gwangjin.github.io/**

광진구는 서울특별시 동부, 한강 북안에 자리한 자치구로 1995년 성동구에서 갈라져 나온 비교적 젊은 행정구역입니다. 건국대학교와 세종대학교를 낀 건대입구 일대가 서울 동부를 대표하는 대학 상권이자 젊음의 거리로 성장하면서 밤낮없이 활기가 넘치는 곳입니다. 능동의 서울어린이대공원, 광장동의 아차산, 자양동의 뚝섬 한강변이 한 구 안에 모여 있어 도심 속에서 산과 강, 대형 공원을 동시에 누릴 수 있는 드문 생활권을 이룹니다. 구의역 앞으로 옮겨온 새 통합청사와 함께 재개발이 이어지며 주거·상업·문화 기능이 촘촘히 얽힌 도시로 자리잡았습니다. 강변역 곁의 동서울종합버스터미널은 수도권 동부와 강원 방면을 잇는 관문 역할을 오랫동안 맡아 왔습니다.
